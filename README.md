# Link Hub 设计规范

本文件是 Link Hub 项目的统一视觉规范。以后新增页面、组件或改版前，必须先阅读本文件，并优先复用这里的颜色、字体、间距、圆角、阴影、边框和动效规则，确保整个项目风格一致。

## 1. 配色

### 核心色

| 角色 | 色值 | 用途 |
|---|---:|---|
| 页面背景 | `#f7f4ee` | 全站主背景、侧边栏背景、开始页背景 |
| 页面外层参考背景 | `#d8d2ca` | 仅作为外层氛围参考，不用于当前全屏主界面 |
| 主文字 / 墨色 | `#1f222b` | 标题、主按钮文字、重要图标 |
| 正文文字 | `#242424` | 正文、常规信息 |
| 次级文字 | `#74716a` | 描述、标签、辅助说明 |
| 卡片背景 | `#f9f9f9` | 链接卡片、轻量内容容器 |
| 主强调黄 | `#ffd83d` | 强调状态、品牌点缀，谨慎使用 |
| 柔和强调黄 | `#fff2a7` | 浅强调背景 |
| 侧边栏淡蓝动效 | `rgba(177, 218, 231, 0.26)` | 展开态侧边栏按钮 hover 填充 |
| 侧边栏选中淡蓝 | `rgba(177, 218, 231, 0.4)` | 展开态侧边栏按钮 active 填充 |
| 收起态圆环 | `rgba(31, 34, 43, 0.1)` | 收起态侧边栏图标选中/hover 圆环 |

### 边框、阴影与透明色

- 常规边框：`rgba(39, 37, 32, 0.12)`
- 轻边框：`rgba(39, 37, 32, 0.06)` 或 `rgba(31, 34, 43, 0.08)`
- 顶栏按钮边框：`rgba(39, 37, 32, 0.1)`
- 毛玻璃菜单背景：`rgba(255, 255, 255, 0.68)`
- 毛玻璃菜单边框：`rgba(255, 255, 255, 0.56)`
- 基础阴影：`rgba(39, 37, 32, 0.12)`
- 深色柔和阴影：`rgba(31, 34, 43, 0.08)` 到 `rgba(31, 34, 43, 0.14)`

### 使用原则

- 默认采用克制的浅色产品 UI，不使用大面积高饱和背景。
- 页面、侧边栏、开始页背景必须统一使用 `#f7f4ee`。
- 淡蓝色只用于导航状态、按钮填充和微交互反馈，不作为大面积背景。
- 黄色只用于重点强调，不作为普通按钮默认色。
- 灰色文字避免太浅，正文必须保持足够可读性。

## 2. 字体

### 字体族

统一使用无衬线系统字体，当前代码使用：

```css
font-family: 'Inter', sans-serif;
```

如果新页面需要更稳的系统 fallback，可使用：

```css
font-family: Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
```

### 字号层级

| 层级 | 字号 | 字重 | 行高 | 用途 |
|---|---:|---:|---:|---|
| 开始页大标题 | `clamp(48px, 9vw, 116px)` | `800` | `0.9` | 首页入口品牌标题 |
| 页面标题 | `24px` | `800` | `1.2` | 顶栏页面标题 |
| 开始页介绍 | `15px` | `400` | `1.8` | 简短说明文字 |
| 侧边栏展开项 | `15px` | `700` | `1` | 分类导航文字 |
| 顶栏按钮 | `13px` | `700` | 默认 | 工具按钮、下拉触发器 |
| 卡片正文 | `14px` 左右 | `400-600` | `1.4-1.6` | 卡片描述、辅助内容 |
| 标签/胶囊说明 | `12px` | `700` | 默认 | 开始页标签、轻量状态 |
| 移动端/图标按钮 | `14px` | `600-700` | 默认 | 紧凑控件 |

### 字体规则

- 产品界面不使用装饰字体。
- 标题不使用负字距，当前规范统一 `letter-spacing: 0`。
- 按钮文字使用较高字重，保持清晰、稳定。
- 长说明文字行高不低于 `1.6`，开始页说明使用 `1.8`。

## 4. 组件样式

### 开始页

- 背景：`#f7f4ee`
- 布局：`position: fixed; inset: 0; display: grid; place-items: center`
- 层级：`z-index: 3000`
- 装饰：允许使用极轻的圆环线和水平细线，颜色透明度低。
- 退出动效：`opacity` 淡出 + `transform: scale(1.5)` 放大。
- 退出时长：`1.35s cubic-bezier(.4,0,.2,1)`

开始页进入按钮：

- 尺寸：`width: min(78vw, 320px); min-height: 80px`
- 内边距：`0 42px`
- 圆角：`999px`
- 边框：`1px solid rgba(31,34,43,0.18)`
- 背景：默认透明
- 字号：`24px`
- 字重：`800`
- hover/focus：淡蓝色从左到右线性填充，`0.72s linear`

### 侧边栏

侧边栏本体：

- 背景必须与主页面一致：`#f7f4ee`
- 展开宽度：`248px`
- 收起宽度：`84px`
- 右边框：`1px solid rgba(39,37,32,0.06)`
- 不使用独立卡片背景，不使用强玻璃效果。

展开态导航按钮：

- 高度：`46px`
- 圆角：`50px`
- 默认背景：透明
- 字号/字重：`15px / 700`
- 不显示分类图标，只显示文字和右侧箭头。
- hover/active 时淡蓝色从 `width: 0` 匀速填充至 `100%`。
- 填充动效：`1.28s linear`
- 文字和图标不得发生位移或缩放。

收起态导航按钮：

- 尺寸：`54px × 54px`
- 圆形：`border-radius: 50%`
- 只显示图标，不显示文字和箭头。
- 不显示蓝色背景，不显示白色图标底。
- hover/active 使用粗圆环反馈。
- 圆环线宽：`5px`
- 圆环颜色：`rgba(31, 34, 43, 0.1)`，active 可加深到 `rgba(31,34,43,0.12-0.14)`
- active 呼吸节奏：`4.8s ease-in-out infinite`

侧边栏展开/收起按钮：

- 尺寸：`56px × 56px`
- 背景：`#f7f4ee`
- 圆角：`18px`
- 边框：`1px solid rgba(39,37,32,0.045)`
- 阴影：`0 18px 32px rgba(31,34,43,0.08)`
- hover 仅增强阴影，不改变背景色。

### 顶端工具栏

- 背景：透明
- 高度：`74px`
- 左侧随侧边栏宽度变化。
- 标题：`24px / 800 / #1f222b`
- 顶栏按钮：
  - 高度：`40px`
  - 内边距：`0 14px`
  - 背景：`rgba(255, 255, 255, 0.78)`
  - hover 背景：`#ffffff`
  - 圆角：`22px`
  - 边框：`1px solid rgba(39,37,32,0.1)`
  - 阴影：`0 8px 18px rgba(39,37,32,0.05)`
  - 字号/字重：`13px / 700`

### 下拉菜单

- 必须出现在对应工具按钮正下方。
- 定位：`position: absolute; top: calc(100% + 10px); left: 50%`
- 变换：关闭 `translate(-50%, 10px) scale(0.98)`，打开 `translate(-50%, 0) scale(1)`
- 最小宽度：`180px`
- 背景：`rgba(255,255,255,0.68)`
- 毛玻璃：`backdrop-filter: blur(18px) saturate(1.35)`
- 圆角：`16px`
- 边框：`1px solid rgba(255,255,255,0.56)`
- 阴影：`0 18px 40px rgba(31,34,43,0.14)`
- 内边距：`8px`

### 卡片

- 背景：`#f9f9f9`
- 圆角：`12px`
- 内边距：`15px`
- 阴影：`0 4px 12px rgba(39, 37, 32, 0.12)`
- 最大宽度：`220px`
- 网格：`repeat(auto-fit, minmax(180px, 1fr))`
- 网格间距：`25px`
- hover 可轻微上浮，但不要过度弹跳。

## 5. 动效规则

### 基础动效

- 全局常规过渡：`0.3s`
- 顶栏按钮：`0.2s ease`
- 侧边栏展开按钮填充：`1.28s linear`
- 侧边栏箭头移动：`1.28s cubic-bezier(.16,1,.3,1)`
- 开始页退出：`1.35s cubic-bezier(.4,0,.2,1)`

### 动效原则

- 产品 UI 动效只服务于状态反馈，不做装饰性炫技。
- 不要让按钮文字在 hover/active 时晃动、缩放或跳动。
- 可动画属性优先使用 `opacity`、`transform`、`width`、`box-shadow`。
- 所有动画必须兼容 `prefers-reduced-motion: reduce`。

## 6. 新页面执行规则

1. 新建或改造任何页面前，先阅读本 `design.md`。
2. 优先复用本规范中的 CSS 变量、尺寸、圆角、阴影和动效。
3. 页面背景、侧边栏背景和开始页背景必须保持 `#f7f4ee`。
4. 新增导航、按钮、卡片、下拉菜单时，必须匹配本文件对应组件规则。
5. 不新增与本规范冲突的强渐变、大面积高饱和色、过重玻璃、夸张阴影或过度圆角。
6. 如果确实需要新增样式 token，必须补充到本文件后再使用。
